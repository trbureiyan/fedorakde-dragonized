My customization of my Fedora Linux 38 KDE Plasma. Small tutorial about the themes I use, programs, and more.

In this repository you'll find:

- fish & starship prompt
- Kvantum themes
- Plasma color (Sweet)
- Plasma Desktop theme (Sweet-Dark)
- Wallpapers
- Latte Dock profiles
- Plasma Widgets

- Cursor skin / style
- Plasma Effects
